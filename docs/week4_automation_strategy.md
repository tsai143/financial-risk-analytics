\## Execution Method

\- Python notebooks are executed automatically using nbconvert

\- Windows Task Scheduler manages execution frequency

\- Logs are generated for monitoring refresh success



\## Limitations

\- Market data refresh is simulated for internship scope

\- Can be extended to real APIs (Yahoo Finance, Alpha Vantage)



\## Production Readiness

\- Scheduler-based automation mimics enterprise batch jobs

\- Design supports migration to cloud schedulers or CI/CD pipelines
